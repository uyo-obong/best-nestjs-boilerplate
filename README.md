## Running the app

```bash
# Step 1.
$ Create .env with your database connection url e.g DATABASE_CONNECTION=mongodb://catalog_mongodb/citisquare_catalog

# Step 2.
$ npm install

# Step 3.
$ docker-compose up --build


## PS: Typeorm mongodb is the database driver

```
